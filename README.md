## 🏠 Executar localmente

Clone o projeto

```bash
git clone https://github.com/FabioPenedo/taskflow-frontend
```

Entre na pasta do projeto

```bash
cd taskflow-frontend
```


Instale as dependências

```bash
npm install
```

Inicie o servidor

```bash
npm run dev
```
